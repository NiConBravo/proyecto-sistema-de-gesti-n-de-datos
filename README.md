Este código forma parte del proyectop final del módulo "Introducción a la programación con python" y corresponde a un sistema de administración de inventario, desarrollado en Python, que permite gestionar libros mediante operaciones de entrada, salida, ajuste y consulta de stock, manteniendo un historial de movimientos (Kardex).
Se desarrolló con programación asistida por IA en el entorno de desarrollo Antigravity. Esto permitió incorporar habildiades de estructuración de documentos e importación de mmódulos.
El desarrollo del proyecto se realizó siguiendo un organigrama de ejemplo proporcionado en el curso, el cual define la separación lógica de responsabilidades dentro del sistema.

Organización del código:
	Cada bloque funcional del sistema fue implementado como un archivo independiente
	Los nombres de los archivos y funciones fueron elegidos deliberadamente para asimilar la estructura del organigrama
	Esto facilita la lectura, el mantenimiento y la relación directa entre diseño conceptual y código
	El objetivo principal fue que el código reflejara claramente la lógica del diagrama, más que optimizar o compactar en exceso.
